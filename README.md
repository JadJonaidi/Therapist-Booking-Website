# Therapist Booking Website

Full-stack therapy practice website featuring consultation booking, admin-managed availability, Express API routes, and SQLite persistence.

---

## Overview

This project is a responsive website for a therapy practice. It includes informational pages, service descriptions, and a prototype booking system supported by a backend server.

---

## Features

- Responsive frontend design  
- Consultation booking flow  
- Admin-managed availability  
- Express backend API  
- SQLite data storage  
- Email notifications  

---

## Tech Stack

- JavaScript  
- HTML / CSS  
- Node.js  
- Express.js  
- SQLite  

---

## Setup

```bash
npm install
npm run dev
```

Open:
```
http://localhost:3000
```

---

## Notes

This is a prototype full-stack project demonstrating backend APIs, booking logic, and database integration.
